Exit / Enter Countries

Here are all the European countries that import and export natural gas. The size of each circle is proportional to the amount of natural gas traded. We can see right away the major importers (Germany) and exporters (Norway) of natural gas.

To see the importers and exporters more clearly, the circles are grouped based on import / export.

Here we overlay natural gas import / export on top of GDP, and we can see that there isn't a very distinct relationship. For example, wealthy countries don't necessarily import more natural gas.

This networks shows which countries have trading relationships for natural gas, some varying by year. For example, Norway, Germany and Russia exports to multiple countries while Iran only exports to Turkey.

The import / export countries are sorted by the gas traded that particular year.
